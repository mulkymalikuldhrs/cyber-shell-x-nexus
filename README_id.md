## Gambaran Umum

**CyberShellX Nexus** adalah platform keamanan siber tingkat lanjut yang mengintegrasikan kecerdasan buatan untuk memberikan panduan keamanan secara real-time, demonstrasi alat interaktif, dan skenario pelatihan edukatif. Dibangun dengan stack TypeScript/React modern dan ditenagai oleh Google Gemini AI dengan sistem fallback multi-API yang andal, platform ini menyediakan pendamping cerdas bagi profesional keamanan siber, mahasiswa, dan penggemar untuk analisis jaringan, penilaian kerentanan, metodologi pengujian penetrasi, forensik digital, dan banyak lagi.

Platform beroperasi melalui tiga antarmuka berbeda: terminal baris perintah untuk pengguna berpengalaman, dasbor web berbasis browser untuk eksplorasi interaktif, dan asisten suara Android native untuk konsultasi keamanan saat bepergian. Setiap antarmuka terhubung ke backend berbasis AI yang sama, memastikan respons yang konsisten dan diperkaya terlepas dari metode aksesnya. CyberShellX Nexus menekankan peretasan etis dan pengungkapan yang bertanggung jawab, menyematkan pemberitahuan hukum dan pedoman etis dalam setiap interaksi.

Proyek ini merupakan bagian dari ekosistem keamanan siber yang lebih luas bersama [HermesQuantOS](https://github.com/mulkymalikuldhrs/HermesQuantOS), berbagi pola arsitektur dan strategi integrasi AI untuk perkakas keamanan generasi berikutnya.

## Fitur

### Asisten Keamanan Siber Bertenaga AI
- **Integrasi Multi-API Gemini** -- Fallback otomatis antara 4 endpoint Gemini API yang memastikan layanan tanpa gangguan bahkan ketika API individual mengalami downtime atau pembatasan tarif
- **Pemrosesan Perintah Cerdas** -- Analisis konteks-aware terhadap perintah keamanan siber yang mencakup pemindaian jaringan, penilaian kerentanan, eksploitasi, forensik, keamanan nirkabel, dan kriptografi
- **Respons AI yang Ditingkatkan** -- Respons dasar diperkaya dengan analisis Gemini AI real-time, menambah kedalaman teknis, contoh praktis, dan praktik terbaik terkini
- **Pemantauan Kesehatan API Real-Time** -- Pelacakan status langsung dari semua endpoint API dengan peralihan otomatis saat mendeteksi kegagalan
- **Mesin Pedoman Etis** -- Setiap respons menyertakan pemberitahuan hukum yang sesuai dan pengingat pengungkapan yang bertanggung jawab

### Antarmuka Baris Perintah (CLI)
- Terminal keamanan siber interaktif dengan integrasi AI
- Simulasi alat: nmap, metasploit, wireshark, sqlmap, burpsuite, hashcat, aircrack-ng
- Demonstrasi perintah edukatif dengan penjelasan terperinci
- Kompatibilitas lintas platform (Linux, macOS, Termux di Android)
- Klasifikasi tingkat kesulitan (pemula, menengah, lanjutan)

### Dasbor Web
- Aplikasi React 18 + TypeScript + Vite modern
- Simulasi terminal real-time dengan dukungan WebSocket
- Demonstrasi alat keamanan siber dan skenario interaktif
- Integrasi database PostgreSQL dengan Drizzle ORM
- Desain responsif dengan tema gelap profesional
- Pustaka komponen Radix UI dengan gaya shadcn/ui

### Asisten Suara Android
- Asisten yang diaktifkan suara dengan deteksi kata pembangun "Hey CyberShell"
- Layanan latar belakang untuk fungsionalitas selalu aktif
- Kemampuan kontrol sistem (WiFi, Bluetooth, senter, volume)
- Antarmuka eksekusi perintah shell
- Panduan keamanan siber berbasis AI melalui percakapan alami

### Skenario Pelatihan Interaktif
- Pemula: Penemuan jaringan, latihan analisis log
- Menengah: Pengujian aplikasi web, penilaian keamanan nirkabel
- Lanjutan: Simulasi APT, investigasi forensik memori
- Prompt pembelajaran yang mencakup OWASP Top 10, fase pengujian penetrasi, pengungkapan yang bertanggung jawab

## Instalasi

### Prasyarat
- **Node.js** 18+ dan npm
- Database **PostgreSQL** (untuk antarmuka web)
- **Git** untuk kloning dan pembaruan

### Mulai Cepat

```bash
# Klon repositori
git clone https://github.com/mulkymalikuldhrs/cyber-shell-x-nexus.git
cd cyber-shell-x-nexus

# Instal dependensi
npm install

# Siapkan skema database
npm run db:push

# Luncurkan dengan menu interaktif (disarankan)
./launcher.sh
```

### Mode Peluncuran

```bash
./launcher.sh              # Menu interaktif (disarankan)
./launcher.sh cli          # Shell keamanan siber CLI
./launcher.sh web          # Server web di port 5000
./launcher.sh android      # Backend asisten suara Android
./launcher.sh update       # Perbarui sistem dari GitHub
./launcher.sh status       # Pemeriksaan kesehatan sistem
```

### Termux (Android)

```bash
# Unduh dan jalankan installer
curl -o termux-install.sh https://raw.githubusercontent.com/mulkymalikuldhrs/cyber-shell-x-nexus/main/termux-install.sh
chmod +x termux-install.sh
./termux-install.sh

# Luncurkan
cd ~/cyber-shell-x-nexus
./run.sh
```

### Variabel Lingkungan

Buat file `.env` di root proyek:

```env
# Kunci API Gemini (minimal satu diperlukan untuk fitur AI)
GOOGLE_API_KEY=kunci_api_gemini_primer_anda
GOOGLE_API_KEY_2=kunci_api_gemini_sekunder_anda
GEMINI_API_KEY=kunci_api_gemini_alternatif

# Database
DATABASE_URL=postgresql://pengguna:sandi@localhost:5432/cybershellx

# Server
PORT=5000
NODE_ENV=development
```

## Penggunaan

### Terminal CLI

Antarmuka baris perintah menyediakan shell keamanan siber interaktif dengan penjelasan berbasis AI:

```bash
node cli-interface.js
```

Kategori perintah yang tersedia:
- **Keamanan Jaringan**: `scan network`, `nmap`, `wireshark`, `netstat`
- **Penilaian Kerentanan**: `check vulnerabilities`, `nikto`, `openvas`
- **Eksploitasi**: `metasploit`, `sql injection`, `burp suite`
- **Forensik**: `volatility`, `autopsy`, `memory analysis`
- **Nirkabel**: `aircrack`, `wireless security`, `wifi scan`
- **Kriptografi**: `password crack`, `hashcat`, `john ripper`
- **Sistem**: `system info`, `process monitor`, `log analysis`

### Antarmuka Web

Jalankan server web dan navigasikan ke `http://localhost:5000`:

```bash
npm run dev
```

Fitur antarmuka web:
- Terminal interaktif dengan obrolan AI
- Komunikasi WebSocket real-time
- Instalasi dan manajemen alat
- Persiapan lingkungan pengujian penetrasi
- Manajemen sesi dan riwayat perintah

### Aplikasi Android

Bangun dan instal asisten suara Android:

```bash
cd android-assistant
./build-apk.sh
./install-apk.sh
```

## Referensi API

| Endpoint | Metode | Deskripsi |
|---|---|---|
| `/api/command` | POST | Proses perintah keamanan siber dengan peningkatan AI |
| `/api/learning-prompt` | GET | Ambil prompt pembelajaran edukatif acak |
| `/api/scenario/:difficulty` | GET | Dapatkan skenario interaktif berdasarkan tingkat kesulitan |
| `/api/ethics` | GET | Ambil pedoman peretasan etis |
| `/api/ai/status` | GET | Periksa kesehatan API AI dan status fallback |
| `/ws/cybershell` | WS | WebSocket real-time untuk komunikasi terminal |

Lihat [Dokumentasi API](docs/API.md) untuk format permintaan/respons terperinci.

## Arsitektur

Platform mengikuti arsitektur monorepo modular dengan pemisahan yang jelas antara frontend, backend, skema bersama, aplikasi mobile, dan lapisan pemrosesan AI.

- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Express.js + TypeScript dengan dukungan WebSocket
- **Database**: PostgreSQL dengan Drizzle ORM + integrasi Supabase
- **Mesin AI**: Google Gemini 2.5 Flash/Pro dengan sistem fallback 4 endpoint
- **Mobile**: Android native (Kotlin) dengan pengenalan suara dan layanan latar belakang
- **CLI**: Antarmuka terminal Node.js dengan penguraian perintah

Lihat [ARCHITECTURE.md](ARCHITECTURE.md) untuk dokumen desain sistem lengkap.
